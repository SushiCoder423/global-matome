## ステップ1: GitHubアカウントを作る（3分）

**GitHubって何？**  
→ 無料でウェブサイトを公開できるサービス。世界中の人が使ってます。

### やること：

1. https://github.com を開く
2. 右上の **「Sign up」** をクリック
3. メールアドレスを入力
4. パスワードを決める（覚えやすいもの）
5. ユーザー名を決める（例: `yamada-taro`）
6. メール認証のコードを入力
7. 完了！

---

## ステップ2: 新しいリポジトリを作る（2分）

**リポジトリって何？**  
→ サイトのファイルを保管する「フォルダ」みたいなものです。

### やること：

1. GitHubにログインした状態で https://github.com/new を開く
2. **Repository name** に `global-matome` と入力
3. **Public** を選択（無料で使える）
4. **Add a README file** にチェック
5. 一番下の **「Create repository」** ボタンをクリック
6. 完了！

---

## ステップ3: ファイルをアップロードする（3分）

**これが一番重要！**

### やること：

1. 私が作ったファイルをダウンロード（後でリンク送ります）
2. ZIPファイルを解凍（ダブルクリック）
3. 出てきたフォルダの中身を**全部選択**
4. GitHubのページに**ドラッグ＆ドロップ**
5. 一番下に **「Commit changes」** というボタンが出るのでクリック
6. 完了！

---

## ステップ4: サイトを公開する（2分）

**いよいよ世界に公開！**

### やること：

1. GitHubリポジトリのページで **「Settings」** タブをクリック
2. 左側メニューの **「Pages」** をクリック
3. **Source** の下で **「GitHub Actions」** を選択
4. 自動で保存される
5. **完了！**

数分待つと、`https://あなたのユーザー名.github.io/global-matome/` でサイトが見れます。

---

## ステップ5: 自動更新を設定する（オプション）

**毎日自動で新しい投稿を取得したい場合**

### Reddit APIキーを取得:

1. https://www.reddit.com/prefs/apps を開く
2. 一番下の **「are you a developer? create an app...」** をクリック
3. 以下を入力:
   - **name**: Global Matome
   - **App type**: **script** を選択
   - **redirect uri**: `http://localhost:8000`
4. **「create app」** をクリック
5. 画面に表示される情報をメモ:
   - **Client ID**: アプリ名の下に表示される英数字
   - **Secret**: "secret" の右に表示される値

### GitHubに設定:

1. GitHubリポジトリのページで **「Settings」** → **「Secrets and variables」** → **「Actions」**
2. **「New repository secret」** をクリック
3. 2つ追加:
   
   **1つ目:**
   - Name: `REDDIT_CLIENT_ID`
   - Secret: (先ほどメモしたClient ID)
   
   **2つ目:**
   - Name: `REDDIT_CLIENT_SECRET`
   - Secret: (先ほどメモしたSecret)

4. 完了！

これで毎日自動で新しい投稿が追加されます。
